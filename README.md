## Reason JS Sandbox

Simple npm pacakage build env example by facebook/reason

- facebook/reason
- BuckleScript
- Test from Js

## How to use

Install reason https://github.com/facebook/reason#installing-via-opam

```
yarn install
yarn build
yarn test
```

See `src/myMod.re` and `test/js/myMod.test.js`
